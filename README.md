[![License: MIT][mit-image]][mit-url] [![NPM Package][npm-image]][npm-url] [![Github Actions][github-actions-image]][github-actions-url]

## Commands

### Storybook

Run inside another terminal:

```
npm run storybook
```

### Example

```
cd example
npm install
npm start
```

To run tests, use `npm test` or `yarn test`.

## Building

```sh
npm install
npm run build
```
